View on GitHub ▶ Try the Demo

🧒 What is this? (Plain English)

A search engine for holy books that is physically unable to make quotes up: ask about the Bible, Quran, Torah, Gita and more, and every answer comes strictly from verified scripture text with exact verse citations. It also includes TheoBench — a test that grades AI models on how accurately they handle sacred texts.

✨ Key Features

Multi-faith corpus

Verified editions of major scriptures in one queryable corpus.

Zero-hallucination RAG

Answers are built only from retrieved verses — every claim cited.

TheoBench

A benchmark grading LLMs on scriptural accuracy.

Comparative study

Ask one question across traditions side by side.

🕹️ Interactive Demo

Ask a question — every sentence in the answer carries a verse citation. (Illustrative in-browser simulation of what the real system does.)
